# NephroVision Final Defense Workspace

Central workspace for all final-defense materials.

## Folder Overview

- `00_source_materials/` — raw references, datasets, approvals, meeting notes
- `01_project_brief/` — one-page summary and defense storyline
- `02_ieee_report/` — LaTeX IEEE two-column final report
- `03_final_presentation/` — Slidev web-based presentation and exports
- `04_defense_qna/` — anticipated questions and answers
- `05_quality_review/` — consistency checks and overclaim guard
- `06_submission_package/` — final zipped deliverables

## Key Control Files

- `MASTER_CONTEXT.md` — audience, scope, metrics, constraints
- `PROJECT_RULES.md` — writing, terminology, and formatting rules
- `KNOWN_METRICS.md` — verified numbers and benchmarks only
- `DO_NOT_OVERCLAIM.md` — limits, disclaimers, and known weaknesses
