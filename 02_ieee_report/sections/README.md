# Report Sections

One `.tex` file per major section.

## Suggested Files

- `01_abstract.tex`
- `02_introduction.tex`
- `03_related_work.tex`
- `04_methodology.tex`
- `05_results.tex`
- `06_discussion.tex`
- `07_conclusion.tex`

## Instructions

Include each file in `main.tex` using `\input{sections/XX_name.tex}`.
