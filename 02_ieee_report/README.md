# IEEE Final Report

LaTeX IEEE two-column final report for NephroVision.

## Subfolders

- `sections/` — per-section `.tex` files
- `figures/` — diagrams, images, and plots
- `tables/` — standalone table `.tex` files
- `build/` — compiled PDFs and intermediate output

## Main Files

- `main.tex` — report root file
- `refs.bib` — BibTeX bibliography

## Build

Compile `main.tex` with a LaTeX engine that supports BibTeX/Biber.
