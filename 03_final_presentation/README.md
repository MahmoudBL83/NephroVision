# Final Presentation

Slidev web-based final presentation for NephroVision.

## Subfolders

- `slidev/` — Slidev project files
  - `components/` — custom Vue components
  - `public/images/` — static slide images
  - `export/` — PDF and PPTX exports

## Files

- `slide_content_source.md` — slide outline and key points
- `speaker_notes.md` — notes for delivery

## Build

Run `npm run dev` inside `slidev/` to preview.
Export PDF/PPTX when final.
