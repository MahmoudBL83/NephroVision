## Placeholder Behavior

If an image is missing, the slide displays a styled placeholder box with instructions. The presentation compiles and presents correctly without images. Replace placeholders before final defense.

## Style Guide

- Background: dark navy (#0a1628) or transparent
- Accent color: cyan (#00d4ff)
- Text: white (#ffffff)
- Format: PNG (preferred) or SVG
- Resolution: minimum 150 DPI for print quality
